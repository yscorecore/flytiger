### FlyTiger是什么

> FlyTiger 是利用SourceGenerator技术，帮助c#的开发者生成常用代码的一个类库。


<!-- slide -->

### FlyTiger的特性

- 自动生成构造函数
- 快速写单例模式的代码
- 快速事件通知属性
- 最高性能的类型转换

<!-- slide -->

### 如何使用

> ```dotnet add package FlyTiger```
